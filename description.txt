Tutoriel sur le développement avec React.js
(create-react-app)

Requis : 
- node (>=14)
- npm (>=5.2)
- npx (avec npm >=5.2)

1. Création d'un composant stateless (Entete et BoutonNav)
2. Passage de paramètre à un composant (parent-enfant) (props)
3. Création d'un composant stateful (Compteurs)
4. Gestion des événements "standard" (click, change, etc)
5. Gestion des événements "surMesure"
